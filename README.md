# DevOps СИТУ Лабораторная работа №2: Установка Nginx с помощью Ansible

## Описание

Цель лабораторной работы — автоматизировать установку веб-сервера **Nginx** на удаленный хост с помощью **Ansible** и развернуть стартовую страницу с текстом:

> Hello from Ansible

## Структура проекта

```text
.
├── inventory.ini
├── playbook.yml
└── README.md
```

## Содержимое файлов

### `inventory.ini`

Список целевых хостов.

```ini
[webservers]
172.16.10.3 ansible_user=runner
```

### `playbook.yml`

Плейбук выполняет следующие действия:

1. Обновляет кэш пакетов APT.
2. Устанавливает Nginx.
3. Создает файл `/var/www/html/index.html` с содержимым:
   ```
   Hello from Ansible
   ```
4. Перезапускает сервис Nginx при изменении файла.

## Запуск

Выполнить команду:

```bash
ansible-playbook -i inventory.ini playbook.yml
```

## Результат

После успешного выполнения плейбука:

- установлен веб-сервер **Nginx**;
- на целевом сервере создан файл `/var/www/html/index.html`;
- при открытии веб-сервера в браузере отображается сообщение:

```text
Hello from Ansible
```

## Используемые технологии

- Ansible
- Nginx
- Ubuntu/Debian (APT)

## Автор

Лабораторная работа выполнена в рамках изучения автоматизации администрирования с использованием **Ansible**.
